Definition of limit

Limit Properties¶

Make sure that you know the definition of the limit.

Simple Example¶

Let $a \in \mathbb{R}$. We prove that $\lim_{x \to a} x = a$.

Let $f(x) = x$. We need to show that for any open interval $I_y$ centered at $a$, there is some open interval $I_x$ centered at $a$ such that $$ f(\text{any number in $I_x$ except $a$}) \in I_y. $$

Let $I_y$ be any open interval centered at $a$. Now we can choose $I_x$ however we want. We choose $I_x = I_y$, which works, because we want an open interval centered at $a$, and $I_y$ is an open interval centered at $a$. (We need something else for more complicated things than $f(x)=x$.) Now $$ f(\text{any number in $I_x$ except $a$}) \in I_x = I_y. $$ Here we used the fact that $f(x) = x$ for all $x$; if we put something into $f$, it comes out unchanged, so any number in $I_x$ comes out as a number in $I_x$. The proof is done.

If $a$ is any number, then $\lim_{x \to a} x = a$.

If this proof feels difficult, don't give up now; we will find very easy ways to calculate limits later, and that will make math fun again.
